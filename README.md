# Alvarez High School Clock

A digital clock and schedule display system built for Alvarez High School. This web application shows the current time, date, and class schedule in real-time.

## Features

- Real-time digital clock
- Current date display
- Daily class schedule
- Automatic period tracking
- Support for multiple schedule types:
  - Regular Schedule
  - Extended Advisory Schedule
  - Early Release Schedule
- Responsive design for all screen sizes
- School colors theme (Maroon, Grey, and White)

## Technology Stack

- Node.js
- Express
- Socket.IO for real-time updates
- EJS templating
- Modern CSS with CSS Variables
- Responsive Design

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and visit:
   ```
   http://localhost:3000
   ```

## Deployment

The application can be deployed to any Node.js hosting platform (Heroku, DigitalOcean, etc.).
